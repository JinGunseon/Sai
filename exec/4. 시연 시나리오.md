1. **피드 조회**

    - 두 번째 글 : 투표가 포함된 글

2. **상세 보기 - 투표, 좋아요**
    - 두 번째 글의 [ 상세 보기 ] 를 클릭

3. **글 작성 - 사진, 태그**

    - [ 어머니랑 함께한 산책 ] 입력 후 사진 추가 옆 [ + 모양 아이콘 ] 클릭
